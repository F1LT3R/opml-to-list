OPML to HTML List | jQuery Plugin

"OPML to List" is a jQuery Plugin to convert OPML XML documents to HTML Lists.

Author: Alistair MacDonald at SignedOn, Inc. (@SignedOnInc)

Example Code

The following code demonstrates how to use the OPML to List plugin. The jQuery plugin does not handle the AJAX for the user, as the user's AJAX requests may be very specific to the requirements of the application. For example: one user may want to used cached request, another may not. The choice was made not to clutter the jQuery plugin with options.

    $(function(){

      function callback(){
        //console.log( 'OPML to List Finished!' );
      }

      function error(e){
        //console.error( 'AJAX ERROR!' );
      }

      function success(e){
        $('#opml-output').opmltolist(e, callback);
      };

      $.ajax({
          //url      : 'playlist.opml'
          //url      : 'specification.opml'
          //url      : 'publications.opml'
          url       : 'specification.opml'
        , dataType  : 'xml'
        , cache     : false
        , success   : success
        , error     : error
      });

    });
